# Secreto

Send private, end-to-end encrypted notes. Share secrets securely via a one-time link.

**Live service:** [secreto.info](https://secreto.info)

## Features

- End-to-end encryption — the server never sees your content
- Self-destruct after reading
- Expiration timer (1h / 1d / 1w / 1mo)
- Email access gate with OTP verification
- Password protection
- File attachments
- Mobile-friendly

## Based on

Secreto is built on top of [Enclosed](https://github.com/CorentinTh/enclosed) (MIT) with significant additions.
